# NEURAL-NETWORK-MODEL-FOR-STOCK-PREDICTION


## Project Overview

This project predicts stock prices using a Neural Network model built with PyTorch.
The model learns patterns from historical stock price data and predicts future stock values.

The project demonstrates how machine learning can be applied to financial time-series data.

## Technologies Used

Python

PyTorch

Pandas

NumPy

Matplotlib

Scikit-learn

Jupyter Notebook

## Project Files

Stock Predictiom NN.ipynb – Main notebook containing the complete project

trainset.csv – Training dataset

testset.csv – Testing dataset

README.md – Project documentation

## Project Workflow

Load historical stock price data

Preprocess and normalize the dataset

Create time-series sequences for training

Train a Neural Network model

Predict stock prices using the trained model

Evaluate model performance

Visualize predictions and results

## Results

The trained model analyzes previous stock price patterns and predicts future prices.
Graphs are used to compare actual stock prices and predicted prices to evaluate model accuracy.
